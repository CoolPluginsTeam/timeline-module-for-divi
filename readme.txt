== Description ==

**Timeline Module for Divi** is a powerful and easy-to-use plugin that allows you to create stunning timelines on your Divi-powered website. Whether you want to showcase your company's history, tell a story, or highlight important events, this plugin makes it simple and visually appealing.

**Features:**
* Fully compatible with Divi Builder.
* Create vertical both sided and single sided timelines.
* Customize colors, fonts, and styles to match your website.
* Add images, videos, and icons to your timeline events.
* Responsive design ensures your timelines look great on all devices.
* Easy to use with a user-friendly interface.

== Installation ==

1. Upload the `timeline-module-for-divi` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Divi Builder and add the "Timeline" module to your page.
4. Customize the timeline settings and start creating your timeline.

== Frequently Asked Questions ==

= How do I add a timeline to my Divi page? =
After activating the plugin, go to Divi Builder, and you will find a new "Timeline" module. Add it to your page and customize it as per your requirements.

= Can I customize the look of my timeline? =
Yes, you can customize colors, fonts, styles, and more using the module settings in Divi Builder.

= Is the plugin responsive? =
Yes, the timelines created with this plugin are fully responsive and will look great on all devices.
