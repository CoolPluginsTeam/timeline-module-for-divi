== Description ==

Timeline Module For Divi allows you to create beautiful timelines in your Divi Theme Builder. This module is perfect for showcasing events, milestones, or any other chronological content in a visually appealing way.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/timeline-module-for-divi` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the Divi Builder and add the Timeline Module to your page.

== Frequently Asked Questions ==

= How do I add a timeline to my Divi page? =

After activating the plugin, open the Divi Theme Builder on your desired page and add the "Timeline Module For Divi" module from the module list.

= Can I customize the appearance of the timeline? =

Yes, the module includes various customization options such as color, layout and images to fit your design needs.
